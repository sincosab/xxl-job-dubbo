## Introduction
XXL-JOB作者认为xxl-job不应该为了支持dubbo而引入注册中心这么重的东西，这样会与产品轻量级的设定产生冲突。

不过还是有许多公司使用dubbo作为rpc框架，如果想要xxl-job还需要另外再对服务提供方增加一些xxl-job-executor依赖。

本项目基于1.9.2版本做的二次开发，能够将调用现有的dubbo服务与xxl-job的其他功能结合。
